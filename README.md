# 🗓️ Notion D-Day Widget

[youngdev57/miny-widget](https://github.com/youngdev57/miny-widget) 프로젝트를 Fork 하여 D-Day 위젯 기능을 수정한 저장소입니다.

## ✨ 주요 추가 기능

기존 소스 코드에 다음과 같은 편의 기능을 보완하였습니다.

- **📅 기념일 계산 옵션 추가**: 기념일 당일을 1일로 포함하여 계산할 수 있는 옵션을 추가했습니다. (inclusive 파라미터를 true로 설정)
- **🔢 날짜 형식 자동 변환**: date 파라미터가 `2025021`와 같이 넘어와도 `2025-02-01` 형식으로 자동 변환하여 표시합니다.
